Для повторного использования сборки нужно перенести папку src, gulpfile.js И package.json в новый проект.

Для поиска обновлений всех модулей использовать команду:
ncu -u --packageFile package.json

Для установки всех модулей, то есть самой сборки:
npm i

Для запуска проекта, то есть локального сервера и сборки всех файлов в dist:
gulp

npm i webp-converter@2.2.3 --save-dev
